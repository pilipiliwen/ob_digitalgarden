# markdown语法
## 文字编辑

1. ##标题
	"#"由少到多，标题由大到小
	"#"后不加空格代表标签：
	#标签
	
2. **加粗**
	文字前后分别加两个星号**

3. *斜体*
	文字前后分别加一个星号*

4. ~~删除线~~
	文字前后分别加两个波浪线~~

5. ==高亮==
	文字前后分别加两个等于号==
